This repository contains the Python code and output files used for COMP4037 Coursework 2: Data Visualization.

## Files
- `data_preprocessing.py`: cleans the original NHS spreadsheet and creates `clean_wide.xlsx` and `long_data.xlsx`
- `plot_heatmap.py`: generates the final matrix heatmap from `long_data.xlsx`
- `original_dataset.xlsx`: source data file
- `matrix_heatmap.png`: final visualisation output

## How to run
1. Run `data_preprocessing.py` to create `clean_wide.xlsx` and `long_data.xlsx`
2. Run `plot_heatmap.py` to generate `matrix_heatmap.png`

## Output
The final figure is saved as `matrix_heatmap.png`.
